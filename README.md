## 📚 Sobre o Repositório
Este repositório armazena o progresso da disciplina de Programação Orientada a Objetos. O objetivo é consolidar o domínio da sintaxe Java e a aplicação de boas práticas de desenvolvimento de software.

## 🛠️ Conceitos Explorados
* **Abstração:** Modelagem de entidades do mundo real.
* **Encapsulamento:** Proteção de dados com modificadores de acesso (getters/setters).
* **Herança:** Reutilização de código e hierarquia de classes.
* **Polimorfismo:** Sobrecarga e sobreposição de métodos.
* **Interfaces e Classes Abstratas.**
* **Tratamento de Exceções.**

## 💻 Tecnologias
* Java JDK 21
* IDE (IntelliJ IDEA / VS Code)
* Git & GitHub
